BibleTime

BibleTime is a powerful cross platform Bible study tool.

BibleTime is a free and open-source desktop program for Bible study. BibleTime may be used to study over 200 free Bible texts, commentaries, dictionaries and books, provided by Crosswire Bible Society via the SWORD library. BibleTime is written in C++ and uses the Qt GUI toolkit.

Downloads

Official downloads for BibleTime can be found on the releases page. Users of Linux and BSD distributions are encouraged to use BibleTime packages provided by their distribution.

BibleTime is not distributed with any Bible translations or other works pre-installed. Please use the Bookshelf Manager in BibleTime to install and update works.

FAQ

Where can I write down my personal notes?
You have to install the module "Personal", which is available as a commentary on Crosswire. Please refer to BibleTime's handbook for more information about the personal commentary.
Why is one of the two testaments of a Bible module empty?
Some modules only contain the text of the old or new testament.
